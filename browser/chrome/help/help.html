Getting Started

Ibid is a citation manager that runs entirely in your browser. All processing happens locally via a Rust/WebAssembly engine — no data is sent to any server.

  • Popup — Click the Ibid icon on any page to cite it
  • Side Panel — Click "Library" for full citation management
  • Context Menu — Right-click for quick cite options
  • Settings — Gear icon for style, locale, theme

Citing a Webpage

  1. Navigate to the page you want to cite.
  2. Click the Ibid icon (or Ctrl+Shift+C).
  3. Fields auto-fill from structured data (JSON-LD, Highwire Press, Dublin Core, OpenGraph). If a DOI is found and fields are missing, auto-enhance fills them.
  4. Review, edit fields, choose citation style.
  5. Click Copy for bibliography or in-text citation.
  6. Click Add to Project to save.
Tip: On restricted pages (chrome://), the form opens empty for manual entry. Paste a DOI and click Enhance.

Auto-Enhance & Identifier Lookup

Automatic

When a DOI/PMID/arXiv ID is found and key fields are empty, Ibid silently resolves via API and fills them. Enhanced fields flash saffron.

Manual (Enhance button)

Paste any identifier and click Enhance:

  • DOI — 10.1038/nature12373 → CrossRef
  • ISBN — 978-0-13-468599-1 → Open Library
  • PMID — PMID:12345678 → NCBI/PubMed
  • arXiv — arxiv:2301.07041 → arXiv API

Enhance never overwrites fields you've already filled.

Projects & Per-Project Styles

Click + next to the project selector to create a project with its own citation style.

Manual override > Project style > Global default

A badge shows Project or Global to indicate which style is active. Switching projects auto-switches the style.

Library Management

Click Library in the popup header to open the side panel:

  • Search — by title, author, DOI, type, journal, keyword
  • Star — click star icon to mark favorites
  • Select — checkboxes for bulk operations (Star, Unstar, Delete)
  • Edit — click any citation to open edit panel
  • Delete — X icon or bulk delete

Editing Citations

Click a citation in the library to edit all fields. Author format: Last, First; Last, First. Date format: YYYY-MM-DD or YYYY.

Edits are permanent and never revert to auto-extracted values.

Importing Citations

Side panel → Import tab. Three methods:

  • Drag & drop .bib, .ris, or .json files
  • Browse files via file picker
  • Paste BibTeX, RIS, or CSL-JSON (format auto-detected)

Preview shows parsed entries with checkboxes for selective import. Duplicates auto-skipped.

Exporting Citations

Side panel → Export tab. Formats:

BibTeX (.bib)
RIS (.ris)
CSL-JSON (.json)
Text (.txt)
HTML (.html)
Markdown (.md)

Scope: All or Starred. BibTeX: toggle abstract/keywords. Text/HTML/Markdown: choose citation style. Download or copy to clipboard.

Duplicate Detection

Checks on add and import. Match criteria:

  • DOI (case-insensitive)
  • ISBN (ignoring hyphens/spaces)
  • URL (exact, excluding search engines)
  • Normalized title (letters+digits only, >10 chars)

Popup: confirm dialog. Import: silently skipped with count.

Quick Settings

Click the gear icon in the popup header:

  • Default Style — global fallback (all 9 bundled editions)
  • Locale — language for terms (English, Spanish, French, etc.)
  • Theme — Light / System / Dark
  • Help — opens this page
  • Advanced Settings — full options page

Backup & Restore

From Advanced Settings:

  • Backup — downloads JSON with all data
  • Restore — upload a backup file
  • Clear All — deletes everything (irreversible)

Privacy

  • Zero telemetry, zero tracking
  • All processing local (Rust/WASM)
  • No bandwidth abuse
  • Minimal permissions: activeTab, storage, sidePanel, contextMenus
  • Network calls only when you trigger Enhance
  • No account required

Troubleshooting

Empty form on page open

Content script can't inject on chrome://, extension pages, or some restricted sites. Use manual entry or paste a DOI.

Author missing

Paste the DOI and click Enhance. CrossRef usually has the correct author list.

Fallback citation format

WASM may still be loading. Wait a moment and change the style to re-render.

Extension not working after update

Go to chrome://extensions/ and click the reload icon on Ibid.
